OpenSUSE Build Service Hosting



  1. Create / update the <package-name>.spec.in file.

    1. Ensure version and release fields are updated.

  2. Delete old source tarball.

  3. If Debian packaging is being created / hosted:-

    1. Delete the following obsolete files:-

      • control file

      • rules file

      • *.orig.tar.gz

      • *.debian.tar.gz

      • *.dsc

    2. Copy in the latest version of the above files.

  4. Download the latest source tarball.

  5. Build the package locally at least once to ensure *.spec.in refreshes *.spec. e.g.

    1. osc-build-local-package.sh

      • Will build openSUSE Leap default.

  6. Add / remove files within osc.

    1. osc ar

  7. Update the osc ChangeLog.

    1. osc vc

  8. Commit the package to the openSUSE Build Service

    1. osc commit
