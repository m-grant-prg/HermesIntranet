Private Hosting Of Debian Package



  1. Change to the pbuilder output directory.

  2. Copy the Debian file to the apt repository host. e.g.

      • scp *.deb mgrant@hermes:staging

  3. Install the Debian file in the apt repository.

    1. Logon to the apt repository hosting machine

      • ssh mgrant@hermes

    2. Move to the directory used above.

      • cd staging

    3. Load the Debian file to the repository for distribution stretch and component beta.

      • sudo reprepro -vb /srv/repos/package/apt/debian -C beta includedeb stretch *.deb

    4. Remove the staged file.

      • rm -v *
