Java Package Creation Guide



  1. Update git.

    1. Ensure development branch commits are up-to-date.

    2. Push development branch.

      • git push

    3. Merge development branch into master.

      • git checkout master

      • git merge mg

      • git push

    4. Create release tag.

      • git tag -a v1.0.1 -m 'Release of version 1.0.1'

      • git push â€“tags

    5. Return to development branch.

      • git checkout mg

  2. Complete Java actions in Netbeans.

    1. Clean and build project.

    2. Generate Javadocs.

  3. Make the Java tarball.

    1. Change to project root directory.

    2. ./credisttar

  4. Move Java tarball to deployment server.

    1. Change to tarball directory.

      • cd distribution

    2. Copy tarball to deployment server.

      • scp * mgranthermes@hermes:staging

  5. Move Java tarball to deployment directory on server.

    1. Logon to server.

      • ssh mgranthermes@hermes

    2. Change to staging directory.

      • cd staging

    3. Copy to deployment directory.

      • sudo cp -v * /srv/deployments

    4. Clean-up.

      • rm -v *
