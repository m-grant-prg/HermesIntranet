rpm Package Preparation Guide



  1. Create / update the <package-name>.spec.in file.

    1. Ensure version and release fields are updated.

  2. Delete old sources.

  3. Download the latest source tarball.

  4. Add / remove files within osc.

    1. osc ar

  5. Update the osc ChangeLog.

    1. osc vc

  6. Build the package locally. e.g.

    1. osc-build-local-package.sh

      • Will build openSUSE Leap default.

    2. osc-build-local-package.sh Fedora_29

      • Will build Fedora 29.
