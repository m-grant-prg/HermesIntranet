General Deployment Installation Guide



  1. If there is an already installed version being superseded:-

    1. Change to the installation source directory.

      • cd ~/Deployments/agmaint-v1.0.8

    2. As root or via sudo

      • make uninstall clean

    3. Clean-up.

      • Change to the parent directory.

        • cd ..

        • rm -rI agmaint-v1.0.8*

  2. Download the deployment.

    1. Change to an appropriate deployment directory.

      • cd ~/Deployments

    2. Either download the file via the command line:-

      • For a source tarball.

        • wget http://hermes/deployments/source/agmaint-v1.0.9.tar.gz

      • For a distribution tarball.

        • wget http://hermes/deployments/distribution/agmaint-v1.0.9.tar.gz

    1. Or use a browser:-

      • For a source tarball.

        • Navigate to http://hermes/deployments/source/

      • For a distribution tarball.

        • Navigate to http://hermes/deployments/distribution/

      • Right click the tarball and select 'Save Link As'

      • Select the appropriate deployment directory and click 'Save'.

  1. Extract the tarball.

    1. Either extract via the command line:-

      • tar -xvpzf agmaint-v1.0.9.tar.gz

    2. Or, double click the file in a File Manager window and extract

      • All files.

      • Keep existing directory structure.

  2. Install the deployment.

    1. Change to the extracted directory.

      • cd agmaint-v1.0.9

    2. Install

      • If using a source tarball

        • autoreconf -if

      • ./configure

      • As root or via sudo

        • make install clean
